PineRE

PineRE is an open-source operating system for RK3326-based handheld gaming consoles.

Built upon Alpine Linux, PineRE aims to provide a lightweight, reliable and maintainable operating system that places gaming first while also offering optional desktop, development and recovery environments.

PineRE is developed using an evidence-driven methodology. Every architectural decision is based on observation, investigation and understanding before implementation.

Understand first.

Reference before implementation.

When in doubt, make the architecture simpler before making the code cleverer.

---

Project Goals

PineRE aims to:

- Provide a lightweight operating system for RK3326 handhelds.
- Keep gaming as the primary experience.
- Offer optional desktop, development and recovery environments.
- Build a reliable recovery system capable of repairing PineRE.
- Remain approachable to users and contributors alike.
- Learn from upstream projects while keeping PineRE's own architecture simple and understandable.
